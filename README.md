## Setup


```bash
# Frontend
cd frontend
pnpm dev

# Backend
cd backend
python3 run.py
```

## Environment

Copy the example env files and fill in your keys:

```bash
cp frontend/.env.example frontend/.env.local
cp backend/.env.example backend/.env
```
